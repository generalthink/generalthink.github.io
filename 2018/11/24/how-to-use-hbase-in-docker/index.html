еңЁDockerдёӯдҪҝз”ЁHBase

еҸ‘иЎЁдәҺ 2018-11-24 | жӣҙж–°дәҺ 2020-04-17 | еҲҶзұ»дәҺ HBase | йҳ…иҜ»ж¬Ўж•°пјҡ
жң¬ж–Үеӯ—ж•°пјҡ 1.5k | йҳ…иҜ»ж—¶й•ҝ ≈ 1 еҲҶй’ҹ
жҲ‘д»¬йғҪзҹҘйҒ“HBaseеә•еұӮдҫқиө–HadoopпјҢйӮЈд№ҲеңЁдҪҝз”ЁHBaseзҡ„ж—¶еҖҷеҝ…йЎ»дҝқиҜҒе®үиЈ…Hadoop,иҖҢе№іж—¶жҲ‘д»¬жғіиҰҒеҝ«жҚ·зҡ„дҪҝз”ЁHBaseзҡ„ж—¶еҖҷеҲҷеҸҜд»ҘеҖҹеҠ©dockerжқҘдҪҝз”ЁеҲ«дәәе®үиЈ…еҘҪзҡ„HBaseй•ңеғҸ,иҝҷж ·еӨ§еӨ§зҡ„зңҒж—¶зңҒеҠӣ,иҝҳиғҪжӣҙеҝ«зҡ„еҜ№HBaseжңүдёҖдёӘж·ұе…Ҙзҡ„дәҶи§ЈгҖӮжүҖд»Ҙжң¬ж–Үе°ұд»Ӣз»ҚеҰӮдҪ•еңЁDockerдёӯдҪҝз”ЁHbase

з”ұдәҺhbaseдҫқиө–дәҺhadoop,жүҖд»ҘжғіиҰҒдҪҝз”Ёhbaseеҝ…йЎ»е…ҲиҰҒе®үиЈ…hadoop(дё»иҰҒжҳҜжҲ‘жҮ’),жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮdockerдҪҝз”ЁеҲ«дәәе®үиЈ…еҘҪзҡ„hbase.

е®үиЈ…docker

dockerзҡ„е®үиЈ…еҫҲз®ҖеҚ•,иҜ·еҸӮиҖғhttp://www.runoob.com/docker/centos-docker-install.html

жҹҘзңӢhbaseй•ңеғҸ

йҰ–е…ҲжҹҘзңӢе…ідәҺhbaseзҡ„й•ңеғҸжңүе“Әдәӣ,иҝҗиЎҢ docker search hbase

жҲ‘д»¬е°ұдҪҝз”Ёз¬¬дёҖдёӘеҘҪдәҶ,жҜ•з«ҹstarжңҖеӨҡгҖӮ

жӢүеҸ–й•ңеғҸ

дҪҝз”Ёе‘Ҫд»Ө docker pull harisekhon/hbase е°ұдјҡе°Ҷй•ңеғҸжӢүеҸ–еҲ°жң¬ең°гҖӮз„¶еҗҺжҹҘзңӢеҪ“еүҚжң¬ең°еӯҳеңЁе“Әдәӣй•ңеғҸ

иҝҗиЎҢhbaseе®№еҷЁе№¶иҝӣе…Ҙе®№еҷЁеҶ…йғЁ

  1. docker run -dit harisekhon/hbase

    еҗҜеҠЁе®№еҷЁ,й»ҳи®Өзҡ„зүҲжң¬жҳҜlatest,еҰӮжһңиҝҗиЎҢзҡ„зүҲжң¬дёҚжҳҜйңҖиҰҒеҠ дёҠзүҲжң¬,жҜ”еҰӮdocker run -dit harisekhon/hbase:1.0

  2. docker exec -it dc0716 bash(иҝӣе…Ҙе®№еҷЁеҶ…йғЁ,иҝҷйҮҢзҡ„dc0716жҳҜе®№еҷЁidеүҚеҮ дҪҚ)

еҸҜд»ҘзңӢеҲ°жӯӨж—¶е®№еҷЁеҶ…зҡ„hbaseзӣ®еҪ•

иҝҗиЎҢhbaseе‘Ҫд»Ө

иҝҗиЎҢдәҶ./hbase shellе‘Ҫд»ӨеҗҺе°ұиҝӣе…ҘдәҶhbase shellе‘Ҫд»ӨиЎҢпјҢжӯӨж—¶еҸҜд»Ҙз”Ёе‘Ҫд»Өзҡ„ж–№ејҸжҹҘзңӢhbaseзҠ¶жҖҒ,ж–°е»әиЎЁзӯү
з”ұдәҺжҲ‘д»¬е®үиЈ…зҡ„hbaseй•ңеғҸжҳҜдёҖдёӘдјӘеҲҶеёғејҸзҡ„пјҢеҸӘжңүдёҖдёӘжңҚеҠЎеҷЁпјҢжүҖд»ҘжҹҘзңӢзҠ¶жҖҒжҳҫзӨәеҸӘжңүдёҖдёӘserver

жӣҙеӨҡзҡ„hbaseе‘Ҫд»Өе°ҶеңЁдёӢдёҖзҜҮж–Үз« дёӯи®Іи§Ј

иҝһжҺҘжңҚеҠЎеҷЁзҡ„hbase

hbaseй»ҳи®ӨдҪҝз”Ёзҡ„й…ҚзҪ®ж–Үд»¶жҳҜhbase-site.xmlдҪҚдәҺhbaseе®үиЈ…и·Ҝеҫ„дёӢзҡ„confзӣ®еҪ•пјҢжңүж—¶еҖҷжҲ‘д»¬еҸҜиғҪжІЎжңүжқғйҷҗиҝһжҺҘиҝңзЁӢжңҚеҠЎеҷЁ,дҪҶжҳҜеҸҲжғіиҰҒзңӢhbaseзҡ„ж•°жҚ®жҖҺд№ҲеҠһе‘ўпјҹиҝҷдёӘж—¶еҖҷеҸҜд»Ҙдҝ®ж”№hbase-site.xmlдёӯзҡ„й…ҚзҪ®иҝһжҺҘдёҠиҝңзЁӢжңҚеҠЎеҷЁгҖӮе°ҶйЎ№зӣ®дёӯзҡ„hbase-site.xmlжӣҝжҚўй»ҳи®Өзҡ„hbase-site.xmlз„¶еҗҺйҮҚж–°иҝӣе…Ҙhbase shellеҚіеҸҜгҖӮдҪҶжҳҜйңҖиҰҒжіЁж„Ҹзҡ„жҳҜз”ұдәҺжҲ‘д»¬дҪҝз”Ёзҡ„dockerпјҢжүҖд»ҘжҜҸж¬ЎиҷҡжӢҹжңәе…ій—ӯд№ӢеҗҺеҜ№hbaseе®№еҷЁзҡ„дҝ®ж”№йғҪдјҡиҝҳеҺҹпјҢжүҖд»ҘеҰӮжһңжғіиҰҒдҝ®ж”№д№ӢеҗҺдёҖзӣҙз”ҹж•ҲеҸҜд»Ҙе°Ҷдҝ®ж”№дҝқеӯҳжҲҗеҸҰдёҖдёӘй•ңеғҸгҖӮ

1. жӣҝжҚўиҮӘеёҰзҡ„hbase-site.xml
2. docker commit containerId  saveImageName:version
    жҜ”еҰӮпјҡ docker commit 430d centor-tar:1.0

д№ӢеүҚд№ҹжҸҗеҲ°дәҶhbaseйңҖиҰҒдҫқиө–дәҺhadoopд»ҘеҸҠzookeeper,hbaseиҮӘеёҰдәҶдёҖдёӘzookeeper,еҸҜд»ҘдёҚдҪҝз”Ёе®ғпјҢиҖҢдҪҝз”Ёе·Іжңүзҡ„zookeeperгҖӮжүҖд»ҘиғҪиҝһжҺҘиҝңзЁӢhbaseжңҚеҠЎеҷЁдё»иҰҒжҳҜеӣ дёәжҲ‘д»¬еңЁhbase-site.xmlдёӯй…ҚзҪ®дәҶhadoopд»ҘеҸҠzookeeperзҡ„ең°еқҖ,иҝҷдёӨдёӘй…ҚзҪ®йЎ№зҡ„еұһжҖ§еҰӮдёӢ

иҝҷйҮҢзҡ„localhost:9020йңҖиҰҒжӣҝжҚўжҲҗеҜ№еә”жңҚеҠЎеҷЁдёҠHadoopзҡ„ең°еқҖд»ҘеҸҠз«ҜеҸЈ,йҖҡиҝҮе®ғд»¬зҡ„й…ҚзҪ®жҲ‘д»¬е°ұеҸҜд»ҘеҺ»hbase shellдёҠж„үеҝ«зҡ„ж“ҚдҪңhbaseдәҶгҖӮ
